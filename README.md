# Run and deploy your Lab Studio (Gemini 3) export

This contains everything you need to run your app locally.

> Note: This folder is a snapshot exported from **Lab Studio**, our in-house Gemini 3 workspace. No need to open any external design UIâ€”just run it locally if you need the reference implementation.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
